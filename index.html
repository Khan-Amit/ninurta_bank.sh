🏦 NINURTA BANK ATM

Bitcoin → Thai Baht Bridge | No Swift. No Middlemen.

⚡ Run the ATM on your terminal

$ git clone https://github.com/Khan-Amit/ninurta_bank.git
$ cd ninurta_bank
$ chmod +x ninurta_bank.sh
$ ./ninurta_bank.sh

Then follow the interactive menu to:

  • ✅ Check real BTC balance
  • ✅ Sell BTC for THB
  • ✅ Withdraw to any Thai bank
  • ✅ Generate PromptPay QR
